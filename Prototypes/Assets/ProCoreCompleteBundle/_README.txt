# Installation

The ProCore Bundle is distributed as a set of UnityPackage files.  To install, simply double-click the package you would like
to import.

**Special Note**
	
If upgrading ProBuilder from an existing install, please run the Upgrade Kit prior to opening the new ProBuilder package.

See parabox.co/probuilder/upgrade for detailed install instructions.

# Documentation

Documentation for each tool is available in their respective packages, or on the ProCore website: http://procore3d.com
